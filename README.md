# Laboratorio de Pseudocódigo PSeInt Web

Aplicación web didáctica multipágina para practicar pseudocódigo estilo PSeInt, visualizar algoritmos, recorrer estructuras de control y resolver ejercicios de lógica paso a paso.

Este proyecto integra ejercicios de pseudocódigo en una herramienta interactiva para portafolio.

---

## Objetivo

Crear una herramienta visual para reforzar lógica de programación usando pseudocódigo en español.

El sistema permite:

- Consultar un catálogo de algoritmos.
- Visualizar pseudocódigo estilo PSeInt.
- Ejecutar algoritmos paso a paso.
- Observar memoria de variables.
- Ver salida simulada.
- Visualizar diagrama de flujo.
- Comparar estructuras `Para`, `Mientras` y `Repetir`.
- Evaluar condicionales.
- Generar patrones con asteriscos.
- Practicar con modo examen.
- Copiar equivalentes simples en JavaScript.

## Módulos

### Catálogo

Lista ejercicios organizados por:

- Condicionales.
- Ciclos.
- Números.
- Cadenas.
- Patrones.

### Visualizador

Incluye algoritmos como:

- Par o impar.
- Mayor de tres números.
- Invertir número.
- División por restas sucesivas.
- Comparar palabras por longitud.

Cada algoritmo muestra:

- Pseudocódigo.
- Línea activa.
- Memoria de variables.
- Salida.
- Diagrama de flujo.
- Equivalente simple en JavaScript.

### Ciclos

Compara:

- `Para`
- `Mientras`
- `Repetir / Hasta Que`

Muestra tabla de iteraciones, acumulador y salida.

### Condicionales

Evalúa:

- Relacionales.
- `Y`
- `O`
- `NO`
- `MOD`

### Patrones

Genera:

- Triángulo creciente.
- Triángulo invertido.
- Cuadrado lleno.
- Cuadrado hueco.
- Pirámide.

### Práctica

Modo examen con retroalimentación automática.

Este proyecto puede publicarse como sitio estático desde GitHub Pages.

Pasos generales:

1. Subir el proyecto al repositorio.
2. Entrar a **Settings**.
3. Abrir **Pages**.
4. Seleccionar rama `main`.
5. Guardar.
6. Abrir el enlace publicado.
