# Rescate del material original

El material original contenía ejercicios de pseudocódigo y programación inicial. Para evitar duplicar el repositorio de fundamentos Java, este proyecto se enfoca en el valor distinto del ZIP: pseudocódigo estilo PSeInt y lógica algorítmica.

## Material aprovechado

- Condicionales.
- Uso de MOD.
- Ciclos `Para`, `Mientras` y `Repetir`.
- División por restas sucesivas.
- Invertir números.
- Comparación de palabras por longitud.
- Patrones con asteriscos.
- Trazas de variables.

## Limpieza aplicada

No se integraron:

- Archivos compilados.
- Borradores.
- Configuraciones locales.
- Ejercicios ya cubiertos en el repositorio de Java.
- Archivos ajenos al tema.

## Resultado

Una aplicación web multipágina para practicar pseudocódigo desde una interfaz visual.
