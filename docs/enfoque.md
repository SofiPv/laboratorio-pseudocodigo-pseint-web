# Enfoque del proyecto

Este proyecto presenta ejercicios de pseudocódigo como una aplicación web didáctica orientada a portafolio.

## Contenido integrado

- Condicionales.
- Uso de MOD.
- Ciclos `Para`, `Mientras` y `Repetir`.
- División por restas sucesivas.
- Invertir números.
- Comparación de palabras por longitud.
- Patrones con asteriscos.
- Trazas de variables.

## Organización

La aplicación está separada en módulos para facilitar la navegación:

- Catálogo de algoritmos.
- Visualizador paso a paso.
- Ciclos.
- Condicionales.
- Patrones.
- Práctica.

## Resultado

Una aplicación web multipágina para practicar pseudocódigo desde una interfaz visual, con estructura clara, navegación por temas y documentación lista para GitHub.
