Laboratorio de Pseudocódigo PSeInt

Proyecto web multipágina para practicar pseudocódigo estilo PSeInt, visualizar algoritmos, recorrer estructuras de control, observar memoria de variables y resolver ejercicios paso a paso.

01

Catálogo

Ejercicios organizados por operadores, condicionales, ciclos, cadenas, números y patrones.

 02

Visualizador

Ejecuta algoritmos paso a paso con pseudocódigo, memoria, salida y diagrama de flujo.

 03

Ciclos

Compara Para, Mientras y Repetir con tablas de iteración y condiciones evaluadas.

 04

Condicionales

Evalúa decisiones con Si / SiNo y operadores lógicos aplicados a problemas básicos.

 05

Patrones

Genera figuras con asteriscos y observa el pseudocódigo que controla filas y columnas.

 06

Práctica

Modo examen con retroalimentación de lógica, ciclos, condicionales y trazas.

De pseudocódigo a ejecución visual

Contenido integrado

Pseudocódigo estilo PSeInt

Los ejemplos usan palabras como Proceso, Leer, Escribir, Si, Mientras, Para y Repetir.

Trazas de ejecución

Cada algoritmo muestra línea activa, memoria de variables, salida y resultado final.

Portafolio

El proyecto queda como app web didáctica, separada por módulos y lista para GitHub Pages.

Desarrollado con HTML, CSS, JavaScript y Canvas.
